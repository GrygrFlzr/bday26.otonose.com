# Kanade Birthday Project 2026

A small **Word Train Letter** for Otonose Kanade's 2026 birthday.
It is similar in concept to the story mode of Gartic Phone.

We will be writing three letters (one in English, one in Korean, and one in Japanese) collaboratively for Kanade according to the following rules:

- One person can send a single message per letter
- A single message can only contain up to:
  - EN letter: 5 words per message
  - JP letter: 10 kanji/kana characters
  - KR letter: 20 hangul/hanja characters per message
- Punctuation is not counted toward the word/character limit, and can be placed anywhere in your message, including at the start.
- Messages must connect to the previous messages (aside from the starting message).

For example, an English word train letter could start like this:

```
Gry: Aah the pepeloni, pepeloni. You
```

Then, the next person needs to continue the letter following what's already there:

```
Nadeko: know the pepeloni? The no
```

The contents of your message beforehand do not need to be preplanned, as part of the challenge is coming up with a continuation to what others have written before you.
